## About the Project
This is an indie-developed prototype in the **idle/incremental genre**, featuring **hand-drawn custom graphics**.  
The project focuses heavily on **UI elements**, since in incremental games the interface plays a central role in progression and player experience.  

## Features
- Idle/incremental gameplay loop with progression systems  
- Modular architecture for easy extension and experimentation  
- Custom UI designed specifically for incremental mechanics  
- Hand-drawn art style for a unique and personal visual identity  
- Upgrade system: improve aiming, shooting speed, and gear  
- Crystal-based economy to unlock new abilities and enhancements  

## Notes
- All graphics are **original and hand-drawn**  
- Built with Unity, structured for reusability and modularity  
- Focused on **quick prototyping** and testing incremental mechanics
